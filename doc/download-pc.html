Sd - Downloading for PC Compatibles

Only Sdtty is available for PC compatibles, not Sd.  (Note: summary of availability.)  It runs under DOS.  You can, of course, run it under Windows 3.1, Windows 95, or Windows NT.

You can download the file "INSTALL.EXE" from this Web page onto your computer.  That file is the same as the file that you would get if you obtained the program from us on a floppy disk.

  1. First, be sure that the directory into which you want to download the file exists.  This directory is typically "C:\SD".  If you haven't already created it, create it with these DOS commands:

    > C:
    > MD \SD
    > CD \SD

  2. Then download the file from your Web browser.  The current version is 31.93, released on February 7, 1998.

    - Download Sdtty for PC compatibles.

  3. After downloading, you should have a file named INSTALL.EXE in the directory in which you want Sdtty installed.  Do the actual installation by typing this at the DOS prompt:

    > INSTALL

    This command will unpack and install the program, database, and all of the documentation files into that directory.

  4. You may now delete the file INSTALL.EXE or move it to a floppy as a backup.

  5. If you are installing Sd for the first time and have Windows 3.1, Windows 95, or Windows NT, you probably want to perform a simple operation to make it possible to start the program by clicking an icon.  Instructions for doing this operation are described in the manual.

    If you are upgrading from a previous version, read the Release Notes.

You can also order a diskette and nicely printed documentation from the authors.

IMPORTANT NOTICE:
Versions 31.72 and later behave differently from previous versions, in a way that can cause frustration.  Please read the release notes.
