Sd - Downloading for PC Compatibles

Only Sdtty is available for PC compatibles, not Sd.  (Note: summary of availability.)  It runs under DOS.  You can, of course, run it under Windows 3.1, Windows 95, or Windows NT.

IMPORTANT NOTICE:
There has been a change in file organization, in order to permit people to download just the documentation formats that they want. The file "INSTALL.EXE" used to contain the program, database, and documentation. It now contains only the program and database. If you want the documentation, download one of the other files.

You can download the file "INSTALL.EXE" from this Web page onto your computer.  You can also download the documentation files shown below. All of these files are the same as the files that you would get if you obtained the program from us on a floppy disk. The documentation files contain the same documents, packed into convenient form, that you can browse or download from the Documentation page.

  1. First, be sure that the directory into which you want to download the file(s) exists.  This directory is typically "C:\SD".  If you haven't already created it, create it with these DOS commands:

    > C:
    > MD \SD
    > CD \SD

  2. Then download the file(s) from your Web browser.  The current version is 32.0b, released on September 7, 1998.
    • Download Sdtty for PC compatibles.
    • Download all 4 documents in plain text form.
    • Download all 4 documents in postscript form.
    • Download all 4 documents in Acrobat form.
    After downloading, you should have a file named INSTALL.EXE (and whatever documentation files you downloaded) in the directory in which you want Sdtty installed.  Do the actual installation by typing this at the DOS prompt:

    > INSTALL

    This command will unpack and install the program and database into that directory.

    Doing any or all of these:

    > TEXTDOC
    > PSDOC
    > PDFDOC

    will unpack the requested documentation files.

  3. You may now delete the .EXE files or move them to a floppy as a backup.

  4. If you are installing Sd for the first time and have Windows 3.1, Windows 95, or Windows NT, you will probably want to perform a simple operation to make it possible to start the program by clicking an icon.  Instructions for doing this operation are described in the manual.

    If you are upgrading from a previous version, read the Release Notes.

You can also order a diskette and nicely printed documentation from the authors.

IMPORTANT NOTICE:
Versions 31.72 and later behave differently from previous versions, in a way that can cause frustration.  Please read the release notes.
