Sd - Downloading for PC Compatibles

Only Sdtty is available for PC compatibles, not Sd.  (Note: summary of availability.)  It runs under DOS.  You can, of course, run it under Windows 3.1, Windows 95, or Windows NT.

IMPORTANT NOTICE:
There has been a change in file organization, in order to permit people to download just the documentation formats that they want. The file "INSTALL.EXE" used to contain the program, database, and documentation. It now contains only the program and database. If you want the documentation, download one of the other files.

You can download the file "INSTALL.EXE" from this Web page onto your computer.  You can also download the documentation files shown below. All of these files are the same as the files that you would get if you obtained the program from us on a floppy disk. The documentation files contain the same documents, packed into convenient form, that you can browse or download from the Documentation page.

  1. First, be sure that the folder into which you want to download the file(s) exists.  This folder is typically "C:\SD".  If you haven't already created it, create it with these commands at the Command (MSDOS) Prompt:

    > C:
    > MD \SD
    > CD \SD

  2. Then download the file(s) from your Web browser into that folder.  The current version is 32.4, released on December 6, 1998.

    IMPORTANT NOTICE:
    This version is significantly changed from previous versions. It uses different libraries for its interface with the Windows operating system. This gives much more flexibility (see the release notes and the manual), but there may be problems with this version. In case of difficulty, you can download the previous version (32.0) here.

    If you are running Windows 3.1 or DOS, you must download a different file. Choose the correct one from the list below.

    • Download install.exe which has Sdtty for PC compatibles running Windows 95, 98, or NT.
    • Download installw.exe which has Sdtty for PC compatibles running DOS or Windows 3.1
    • Download textdoc.exe which has all 4 documents in plain text form.
    • Download psdoc.exe which has all 4 documents in postscript form.
    • Download pdfdoc.exe which has all 4 documents in Acrobat form.
  3. After downloading, you should have a file named INSTALL.EXE or INSTALLW.EXE (and whatever documentation files you downloaded) in the folder in which you want Sdtty installed.  Do the actual installation by typing this at the Command Prompt: 

     > install        (or installw)

    This command will unpack and install the program and database into that folder.

    Doing any or all of these: 

     > textdoc
 > psdoc
 > pdfdoc

    will unpack the requested documentation files.

  4. You may now delete the .EXE files or move them to a floppy as a backup.

  5. If you are installing Sd for the first time, you will probably want to perform a simple operation to make it possible to start the program by clicking an icon.  Instructions for doing this operation are described in the manual. If you have Windows 95, 98, or NT 4.0 or later, all you need to do is (still at the Command Prompt) type one of 

     > do_desk             (to put the icon on your Desktop),       or
 > do_start            (to put the icon in the Start Menu),     or
 > do_prog             (to put the icon in the Programs Menu)

  6. If you are upgrading from a previous version, read the Release Notes.

You can also order a diskette and nicely printed documentation from the authors.

IMPORTANT NOTICE:
Versions 31.72 and later behave differently from previous versions, in a way that can cause frustration.  Please read the release notes.
