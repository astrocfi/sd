Sd - Downloading for Unix Workstations

You can download Sd or Sdtty for a number of Unix workstations. These are compressed archives of the programs and documentation. The Unix "tar" command is used for archiving, and "gzip" or "compress" is used for compression. We are changing to using "gzip" for all compression.

IMPORTANT NOTICE:
The file organization is undergoing a change, in order to permit people to download just the documentation formats that they want. The various "tar" files shown immediately below used to contain the program, database, and documentation. We are making new ones that contain only the program and database. (Files ending in ".gz" are the new ones that do not have documentation.) If you want the documentation, download one of the documentation files in the second group. They are independent of the computer and operating system.
